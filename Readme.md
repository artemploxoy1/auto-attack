# Плагин: Авто-кликер и выбор оружия (auto-attack) ⚔️

## Описание плагина

`auto-attack` — это многофункциональный плагин для автоматизации боевых действий и взаимодействия бота с окружением на сервере. В отличие от классической киллауры, данный модуль работает по принципу высокоточного авто-кликера «в одну точку». Бот непрерывно совершает клики строго по направлению своего перекрестия (взгляда), имитируя реальное поведение игрока.

### Ключевые возможности:
* **Имитация ЛКМ и ПКМ:** Бот может как наносить удары (ЛКМ), так и использовать предметы в руке/взаимодействовать с сущностями (ПКМ).
* **Интеллектуальный выбор оружия:** Бот автоматически сканирует инвентарь и вычисляет лучший предмет для атаки по выбранному критерию (урон, прочность, чары) перед совершением удара.
* **Безопасная смена слотов:** Если оружие уже находится на панели быстрого доступа (хотбаре), бот просто бесшумно переключает слот, минуя лаги интерфейса инвентаря и исключая риск случайного попадания оружия во вторую руку (Off-hand).
* **Интеграция с авто-питанием:** Плагин автоматически отслеживает события `autoEat:start` и `autoEat:stop`, приостанавливая клики на время приема пищи, чтобы не сбивать анимации и предотвратить застревание бота.
* **Управление направлением:** С помощью игровой команды вы можете легко скорректировать направление взгляда бота на нужный угол.

## Игровые команды

| Команда | Алиасы | Описание | Аргументы | Пример использования |
| :--- | :--- | :--- | :--- | :--- |
| **head** | `голова` | Поворот головы бота относительно текущего направления взгляда. | `<направление>` `<угол_в_градусах>` | `@head налево 90` <br> `@head вниз 15` |

### Допустимые параметры направления взгляда:
* `налево` — поворачивает взгляд бота влево на указанное число градусов.
* `направо` — поворачивает взгляд бота вправо на указанное число градусов.
* `вверх` — поднимает взгляд бота вверх.
* `вниз` — опускает взгляд бота вниз.

## Настройки плагина в панели

Плагин использует систему условной видимости настроек. При выборе готовых пресетов («Быстрый» или «Медленный») все сложные параметры скрываются для сохранения чистоты интерфейса. Полная конфигурация отображается только при выборе режима **«Вручную»**.

| Настройка | Тип | Описание | Значение по умолчанию |
| :--- | :--- | :--- | :--- |
| **Режим авто-кликера** | `select` | Готовые пресеты задержек: «Быстрый» (400 мс), «Медленный» (1000 мс) или ручная настройка («Вручную»). | `fast` |
| **Тип клика (Кнопка мыши)** | `select` | **ЛКМ** (атакует цель или взмахивает рукой) или **ПКМ** (использует предмет/взаимодействует с целью). | `lmb` |
| **Выбирать лучшее оружие** | `boolean` | Включает алгоритм автоматического поиска и переключения на оптимальный инструмент перед ударом. | `false` |
| **Характеристика для выбора** | `select` | Приоритет при поиске оружия: <br> • *Урон* (базовый урон + острота) <br> • *Прочность* (наибольшая оставшаяся) <br> • *Чары* (сумма уровней всех чар) <br> • *Кастомный слот* (выбор конкретного слота хотбара) | `damage` |
| **Кастомный слот панели** | `number` | Индекс слота на хотбаре (от 0 до 8), на который переключится бот, если выбран кастомный критерий оружия. | `0` |
| **Задержка между атаками (мс)** | `number` | Интервал в миллисекундах между кликами кликера (доступен в ручном режиме). | `600` |
| **Дистанция атаки (блоки)** | `number` | Максимальное расстояние для проверки цели перед перекрестием бота. | `3.5` |

## Права доступа (Permissions)

Плагин автоматически регистрирует в системе следующее право доступа:

*   **`autoattack.head`** — разрешает пользователям использовать команду `@head` (`@голова`) в разрешенных чатах.

По умолчанию это право автоматически добавляется в группу администраторов бота.
